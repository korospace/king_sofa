## Required
<ul>
    <li> NodeJS 16.13.0 </li>
</ul>

## Install
<ol>
    <li> git clone https://github.com/korospace/finjop.git </li>
    <li> npm install </li>
    <li> npm run dev </li>
    <li> open http://localhost:8080 </li>
</ol>
